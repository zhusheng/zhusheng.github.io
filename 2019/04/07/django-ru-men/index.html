Django官方文档

安装Django

# 创建隔离环境
conda create -n django python=3.6
# 安装django最新版
pip install Django
# 查看Django版本
python -m django --version

基本操作

创建项目

创建项目yingcloudiot，会在当前目录下新建一个项目目录yingcloudiot，目录下包含一些文件。

django-admin startproject yingcloudiot

创建app

进入项目路径下，执行如下指令创建一个APP，一个Django项目可以包含多个APP。

python manage.py startapp polls

新建的APP需要在setting.py中进行注册，如下所示：

INSTALLED_APPS = [
    'polls.apps.PollsConfig'
]

启动项目

python manage.py runserver

数据库操作

项目第一次运行，需要执行:

python manage.py migrate

以后每次的数据库更改，执行如下操作:

python manage.py makemigrations 
python manage.py migrate

第一条指令只是记录我们对数据库的更改，缓存到本地，并没有真正的应用。
第二条指令会应用更改。

【拓展】
我们也可以对指定的APP进行数据库更新操作:

python manage.py makemigrations polls
python manage.py migrate polls

python shell

我们可以进入python shell环境，

python manage.py shell

创建超级用户

python manage.py createsuperuser

models字段和属性参考

model数据库操作

在Django项目中，我们涉及到数据库的增删改查操作，我们可以使用django shell来编写测试代码。

首先，进入项目所在路径，执行如下指令：

python manage.py shell

单一表操作

导入需要操作的models

>>> from books.models import Publisher

插入数据

>>> p1 = Publisher(name='Apress', address='2855 Telegraph Avenue',  
...     city='Berkeley', state_province='CA', country='U.S.A.',  
...     website='http://www.apress.com/')  
>>> p1.save()

查询数据

查询全部

>>> Publisher.objects.all()  
[<Publisher: Apress>, <Publisher: O'Reilly>]

查看单个对象

>>> Publisher.objects.get(name="Apress")  
<Publisher: Apress>

说明：如果结果是多个对象或者没有返回结果则会抛出异常。

filter

按字段属性过滤

>>> Publisher.objects.filter(name='Apress')  
[<Publisher: Apress>]

说明：返回的是列表

按字段属性包含内容过滤

>>> Publisher.objects.filter(name__contains="press")  
[<Publisher: Apress>]

order_by

排序

>>> Publisher.objects.order_by("name")  
[<Publisher: Apress>, <Publisher: O'Reilly>]

说明：相当于 order by name asc。

倒序排序

>>> Publisher.objects.order_by("-name")

说明：加个负号相当于 order by name desc。

限制返回数据

>>> Publisher.objects.order_by('name')[0]  
<Publisher: Apress>

说明：相当于 limit 1。

限制返回数据2

>>> Publisher.objects.order_by('name')[0:2]

说明：相当于 OFFSET 0 LIMIT 2。

更新数据

更新1

>>> Publisher.objects.filter(id=52).update(name='Apress Publishing')

更新2

>>> p = Publisher.objects.get(name='Apress') #先查询  
>>> p.name = 'Apress Publishing' #更新  
>>> p.save()  #保存

删除数据

删除1

>>> p = Publisher.objects.get(name="O'Reilly")  
>>> p.delete()

删除2

>>> Publisher.objects.filter(country='USA').delete()
